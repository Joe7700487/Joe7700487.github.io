Project Overview

A fully customized mechanical keyboard featuring 3D-printed components, custom firmware, and professional-grade switches. This project combines hardware design, electronics, and programming to create a unique input device.

Key Features

  • Custom 3D-printed case and plate
  • Arduino Pro Micro controller
  • Hot-swappable mechanical switches
  • Programmable RGB backlighting
  • Custom keymapping support

Technical Details

Build Process

The keyboard was created through several phases:

  • CAD design and 3D printing
  • PCB design and assembly
  • Switch and stabilizer modification
  • Firmware programming
  • Custom keycap selection
